jQueryの詳細説明

$( 'input[name="optionsRadios"]:radio' ).change( function() {

ここでラジオボタンのイベント監視を行っています。
そのあとに

var radioval = $(this).val();

として、選択されたラジオボタンのvalueを変数radiovalに入れています。
ラジオボタンの選択が変わるたびにそのvalueがこの変数に入ってきます。 

if(radioval == 5){

valueが「５」、つまり「その他」が選ばれているときにこのif文を実行するということです。 

if(radioval == 5){
        $('#inputother').removeAttr('disabled');

ここではidのinputotherの要素にある属性「disabled」を削除するという内容になっています。

}else{
            $('#inputother').attr('disabled','disabled'); 
         }

ここではif文の中とは逆に属性を付与しています。そのため、再び要素が無効化されるというわけでです。
