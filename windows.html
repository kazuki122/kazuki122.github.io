windows.open()の使い方

windows.open()とは新たなウィンドウを生成できます。

windows.open(url, ウィンドウ名, [オプション])

第１引数の「url」は生成されるウィンドウに表示するページのURLです。
指定しない場合は「about:blank」の新たなウィンドウを開きます。

第２引数の「ウィンドウ名」は_blankや_selfなどのターゲット属性値などを指定します。
特に指定する必要がなければnullで良いです。

第３引数の「オプション」は新たに開くウィンドウの表示スタイルを指定します。カンマで区切ることにより、複数指定できます。

主に使われるのは以下のパラメータになります。

パラメーター 値 意味
width 数値 ウィンドウ幅
height 数値 ウィンドウ高さ
left 数値 ウィンドウ位置左
top 数値 ウィンドウ位置上
menubar yes/no メニューバーの有無
toolbar yes/no ツールバーの有無
scrollbars yes/no スクロールバーの有無

サンプル

サンプルのソースコード

function sampleOpen() { window.open(
'', '_blank', 'menubar=0,width=500,height=300,top=100,left=100' ); }

windows.close()について

windows.close()はウィンドウを閉じることができます。

サンプル

サンプルのソースコード

function sampleOpen() { sampleWindow = window.open( '', '_blank', 'menubar=0,width=500,height=300,top=100,left=100' ); }
function sampleClose() { sampleWindow.close(); }
